Game – игра

Управление:

A, ← Двигает платформу влево

D, → Двигает платформу вправо

При нахождении курсора мыши в окне управление осуществляется мышью, в противном случае клавишами.

Space – запуск шарика

Реализованы бафы:

R+/R- Диаметр шарика

S+/S- Скорость платформы

P+/P- Длинна платформы

L+    - Плюс жизнь
